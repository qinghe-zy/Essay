# 写作素材池说明

## 目的
写作素材池不是“所有看过的资料集合”，而是“允许后续论文正文直接使用的材料集合”。

它的目的只有两个：

1. 限制正文写作时可用的材料范围
2. 降低空话、假文献、无出处描述进入正文的风险

## 允许进入写作素材池的材料
只有下面材料可以进入后续正文写作：

### 1. 已入 Zotero 的文献
- 中文文献以 CNKI 入库为主
- 外文文献必须先完成核验，再入 Zotero

### 2. 已完成事实卡片的文献
- 不区分中文或外文
- 只要事实卡片里有明确出处、页码或章节定位，就可以进入写作素材池

### 3. 已核验通过的外文文献
- 至少完成一次 Crossref 核验
- 关键字段没有明显冲突

### 4. 真实项目材料
- 项目代码
- 数据库结构
- 页面截图
- 功能说明
- 测试记录
- 接口说明

## 禁止进入写作素材池的材料
- 未核验的外文文献
- 未入 Zotero 的“临时看到的文章”
- 无出处的模型总结
- 没有页码支撑的摘录
- 听起来合理但找不到原始来源的说法

## 最小入池标准
### 中文文献
满足以下条件才能入池：

1. 已进入 Zotero
2. 已人工核对题名、作者、年份、来源
3. 已完成事实卡片

### 外文文献
满足以下条件才能入池：

1. 已完成 Crossref 核验
2. 结论为“可入库”或“基本匹配后人工确认可用”
3. 已进入 Zotero
4. 已完成事实卡片

### 项目材料
满足以下条件才能入池：

1. 材料真实存在
2. 能对应到系统实现、数据库、页面或测试行为
3. 能说明其将在论文哪一章使用

## 入池后怎么用
后续让 Codex 参与论文写作时，只提供下面三类输入：

1. 已入池的文献事实卡片
2. 已入池的项目材料记录
3. 本章要解决的写作目标

不要直接把“还没核验的 PDF、大段网页内容、临时搜索结果”丢进正文写作环节。
