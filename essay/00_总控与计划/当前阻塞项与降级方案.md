# 当前阻塞项与降级方案

| 阻塞项 | 影响范围 | 当前降级方案 |
|---|---|---|
| Zotero MCP 不可用 | 文献入库确认 | 先使用本地文献总台账、候选清单、核验记录和事实卡片 |
| CNKI 可能需人工登录 | 中文文献检索 | 已先录入官网或公开目录页可核验条目，后续继续以 CNKI 可检索来源补齐并回填台账 |
| 总文献数量仍未达最终门槛 | 全文整合 | 继续补中文文献和少量外文，不影响第4章和第5章推进 |
| 缺现成测试记录 | 第5章写作 | 先建立人工测试记录表和定性验证记录，再逐步补实际记录 |
| 构建验证受环境限制 | 运行稳定性表述 | 不写“完整构建验收通过”，只写代码与页面逻辑可验证事实 |
