# 审核通过的执行边界

## 当前确认可写
- 用户账户管理、博客管理、阅读记录、互动功能、评论与评分、消息通知、个人中心、个人维度统计、混合推荐、AI 摘要与标签生成。

## 当前确认只能收缩写
- 数据统计只能写成个人维度统计与画像展示。
- 权限控制只能写成基础访问控制。
- 推荐模块只能写成“混合推荐 + 可选 AI 增强，AI 精排默认关闭”。
- 通知模块当前稳妥口径是“点赞通知和评论通知已实现”。

## 当前明确禁写
- 性能测试、压测、TPS/QPS、响应时间。
- 准确率、召回率、NDCG、MAP、AUC、F1 等推荐量化评估。
- 独立后台管理端。
- 完整 Spring Security / JWT / RBAC 权限框架。
- 预留表与默认关闭能力的已投产描述。

## 当前正文顺序边界
- 先第2章、第3章、第4章、第5章。
- 后第1章、第6章。
- 摘要、英文摘要、致谢、参考文献最后统一处理。
