# 高风险表达回收清单

| 高风险表达 | 当前处理方式 | 原因 |
|---|---|---|
| 完整权限框架 | 统一回收为“基础访问控制” | 项目未实现 Spring Security / JWT / RBAC |
| 推荐效果显著提升 | 删除 | 没有量化评估和对照实验 |
| recommendation 表形成闭环 | 删除 | 未发现稳定读写主链证据 |
| user_behavior 为核心主链表 | 删除 | 当前主业务链主要使用 `visit_log`、`user_like`、`user_action` |
| AI 精排已启用 | 删除 | 配置默认关闭 |
| 收藏通知已实现 | 删除 | 当前只确认点赞、评论通知 |
| 完整性能测试通过 | 删除 | 当前未做性能测试 |
| 后端接口全部自动化通过 | 删除 | 当前自动化测试仅 `contextLoads()` |
| 数据分析平台 | 统一回收为“个人维度统计展示” | 当前仅个人中心概览与画像 |
